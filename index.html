Integraciones de Datos

Este enlace recibe un payload JSON en formato POST cada vez que presionas Registrar.

Recuerda compartir tu planilla como "Cualquier persona con el enlace puede leer" para que el sistema pueda extraer la información.

Código sugerido de Google Apps Script

Copia este código y pégalo en el editor de Apps Script de tu hoja de cálculo para automatizar la inserción.

function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    
    // Append rows sequentially
    sheet.appendRow([
      new Date(),
      data.nombreCliente,
      data.categoria,
      data.fechaMatrimonio || "N/A",
      data.productosStr,
      data.montoTotal,
      data.montoPago,
      data.saldoPendiente,
      data.fechaPago,
      data.tipoEntrega,
      data.fechaEntrega,
      data.comuna || "Retiro"
    ]);
    
    return ContentService.createTextOutput(JSON.stringify({"status": "success"}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({"status": "error", "message": error.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

¿Cómo funciona la sincronización?

1. WhatsApp Directo: Los registros siempre abrirán la pestaña de WhatsApp hacia el número predefinido sin importar la configuración de internet.

2. Google Sheets: Si configuras el Apps Script, la venta se enviará en segundo plano. Así tendrás reportes gráficos automáticos en tiempo real.

3. Catálogo en CSV: Tu planilla de productos debe tener preferentemente la siguiente estructura en las columnas:
SKU | Nombre | Categoria | Precio
